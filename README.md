# Smart Vehicle Safety System

## Overview
The Smart Vehicle Safety System is a Python-based application designed to improve road safety by simulating intelligent vehicle monitoring mechanisms. The system focuses on detecting traffic signals and controlling vehicle behavior in restricted zones to reduce accident risks.

## Problem
Road accidents often occur due to delayed driver response at traffic signals and restricted areas.

## Solution
This project implements a Python-based monitoring logic that simulates traffic signal detection and automatically controls vehicle speed when entering restricted zones.

## Key Features
- Traffic signal detection simulation
- Automatic vehicle speed control
- Safety alerts for restricted zones
- Logic-based road safety monitoring

## Technologies Used
- Python

## How to Run
1. Install Python on your system.
2. Download or clone this repository.
3. Run the program using:
   python vehicle_safety_system.py
