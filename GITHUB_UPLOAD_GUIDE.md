## Step 3: Install Git on Laptop
Open PowerShell:
```powershell
winget install Git.Git
```

## Step 4: Upload Smart Vehicle Safety System
```powershell
cd C:\path\to\smart_vehicle_safety
git init
git add .
git commit -m "Initial commit - Smart Vehicle Safety System"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/smart-vehicle-safety-system.git
git push -u origin main
```
